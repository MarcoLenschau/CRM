# Usage

How to use the CRM application: creating events, managing customers and users, and monitoring activity.

## Creating Your First Event

1. Log in with an admin account
2. Navigate to Dashboard or Calendar
3. Click "Create Event"
4. Fill in event details (name, description, priority, date & time)
5. Save — event appears immediately in the calendar and dashboard

## Managing Customers

1. Go to Customers page
2. Click "Add Customer"
3. Enter Name, email, phone, company, status, assigned user
4. Save to add to the database

## User Administration (Admin Only)

1. Go to Users page (admins only)
2. Create/update/delete users
3. Assign admin privileges as needed

## Monitoring Activity

1. Go to Log page (admin only)
2. Filter by user, action type, entity, or status
3. Investigate audit trail for troubleshooting or review
